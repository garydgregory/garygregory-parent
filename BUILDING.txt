Basic Build Checks
==================

mvn apache-rat:check

mvn verify

Deploy SNAPSHOT locally
=======================

mvn install

Deploy SNAPSHOT to Maven Central
================================

mvn deploy

Releasing
=========

All of the above to check that all is well.

Get a fresh clone:

set REL_VERSION=1.0
set REL_PROJECT=garygregory-parent
del /S/Q %TEMP%\RC\*
mkdir %TEMP%\RC
%SystemDrive%
cd %TEMP%\RC
git clone https://github.com/garydgregory/%REL_PROJECT% .
mvn versions:set -DnewVersion=%REL_VERSION% -DgenerateBackupPoms=false
git commit -am "Update version numbers for release garygregory-parent 1.0"
git tag -s garygregory-parent-1.0 -m "Tag release garygregory-parent 1.0"
git push origin garygregory-parent-1.0
mvn deploy
mvn versions:set -DnewVersion=1.0.1-SNAPSHOT -DgenerateBackupPoms=false
